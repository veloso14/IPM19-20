Boa Interface

A interface em causa é a página principal da aplicação MB Way.

A partir desta página deve ser possível, fácil e intuitivo escolher a operação de transferência de dinheiro que se pretende efetuar. Um aspeto positivo que facilita essa escolha é a forma como as diferentes opções estão dispostas e identificadas.

O facto de os botões serem relativamente grandes, com legendas claras e desenhos intuitivos contribui para que um utilizador, mesmo num primeiro contacto com a interface, em poucos segundos consiga escolher a opção pretendida.

• Ana Rebelo, 51074


Má Interface

A interface em causa é a página de um livro.

Os livros podem ser de diferentes tipos, mas neste caso específico trata-se de um manual escolar. Como tal, o seu conteúdo é de teor informativo e deve ser prático de consultar.

Uma página pode conter texto ou figuras, mas todas elas devem apresentar o número da página.

Neste exemplo, a numeração das páginas é feita no rodapé, com o número centrado. O alinhamento do número faz com que seja necessário abrir mais o livro para que se consiga ver a numeração, tornando o processo de procura mais lento.

Como se trata de um pormenor, é provável que muitos autores prefiram a numeração centrada por uma questão de gosto pessoal ou estética. Mas para o leitor é um pormenor que, embora subjetivo, pode fazer diferença.

Uma melhor alternativa seria alinhar os números à direita. Dessa forma não é preciso abrir tanto o livro e seria mais rápido folhear as páginas até à pretendida.

• Ana Rebelo, 51074

Má Interface

O Clip possuí uma interface pouco apelativa apresentando a mesmo de uma forma pouco intuitiva chegando por vezes a ser confusa.

Por exemplo, para fazer a inscrição num teste, o menu deveria de estar no ano letivo correspondente.

No entanto o menu encontra-se na página do aluno, completamente dissimulado no meio das incrições de turnos, escolares etc...

Para novos utilizadores, torna-se bastante complicado perceber a organização do Clip.
